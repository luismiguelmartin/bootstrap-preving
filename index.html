Preving Bootstrap

Guía de estilos interna para el desarrollo de proyectos y aplicaciones web de Grupo Preving.

Preving Boostrap es el compendio de guía de estilos, biblioteca de componentes HTML+CSS+JS y patrones de diseño front-end para las aplicaciones de Grupo Preving. Está desarroollado sobre el trabajo del proyecto Bootstrap y permite desarrollar de forma rápida aplicaciones gracias al sistema de variables y mixings de SASS, estructura de retícula responsive y los componentes preconfigurados.
