        Utilidades de Maquetación

        Conjunto de utilidades para espaciado, márgenes, bordes y otros

        Espaciado con margin y padding

        Boostrap dispone de clases útiles que nos ayudan en la maquetación de las aplicaciones, jugando con los espacios internos y los márgenes de los elementos. Estas clases se denominan utilizando el formato {propiedad}{lado}-{tamaño}.

        De esta manera, la propiedad puede ser:

        • m - para el margin
        • p - para el padding

        Los lados son:

        • t - para margin-top o padding-top
        • b - para margin-bottom o padding-bottom
        • l - para margin-left o padding-left
        • r - para margin-right o padding-right
        • x - tanto para *-left como *-right
        • y - tanto para *-top como*-bottom
        • en blanco - para las clases que establecen un margin o padding en los 4 lados del elemento (por ejemplo: m-0 o p-0)

        Donde el tamaño es:

        • 0 - Añade un margin o padding de tamaño 0 (sería eliminar el valor de dichas propiedades en el elemento en el que se aplique)
        • 1 - margin o padding de tamaño 1
        • 2 - margin o padding de tamaño 2
        • 3 - margin o padding de tamaño 3
        • 4 - margin o padding de tamaño 4
        • 5 - margin o padding de tamaño 5
        • auto - Para las clases que configuran el margin auto.

        Por ejemplo: pt-3, que correspondería con padding-top y tamaño 3, o my-0, que correspondería con margin-top y margin-bottom establecidos a 0.

        Los tamaños se generan a partir de una variable definida en la configuración de Bootstrap.

        Colores en los bordes

        Se puede cambiar el color del borde empleando los colores definidos en Preving Bootstrap (colores básicos o de aplicaciones).

        Colores de las aplicaciones

        Se puede cambiar el color del borde empleando los colores definidos en Preving Bootstrap (colores básicos o de aplicaciones).

        Del mismo modo se pueden emplear los colores de las aplicaciones. Basta reemplazar la clase de un color (boder-primary, border-warning, etc.), por la correspondiente al color del borde de la aplicación que queremos (por ejemplo: border-app-formacion)
