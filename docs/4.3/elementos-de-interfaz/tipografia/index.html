        También estás disponibles las clases similares de .h1, … , .h6. Están indicadas para emplear el estilo/tamaño de un encabezado pero no se puede utilizar el elemento HTML asociado.

        Elementos de formato inline

        Estilo para elementos de formato HTML5 en línea.

        Texto destacado.

        Marcado como texto borrado o eliminado.

        Texto subrayado

        Texto más pequeño.

        Texto en negrita

        Texto en itálica o cursiva.

        Las clases .mark y .small están disponibles para aplicar los mismos estilos que las etiquetas <mark> y <small>.

        Negritas (<b>) y cursivas (<i>)

        Aunque no se muestran más arriba, se pueden emplear las etiquetas <b> y <i> en HTML5. Así <b> está destinado a resaltar palabras o frases sin transmitir una importancia adicional, mientras que <i> está indicada principalmente para voz, términos técnicos, etc.

        Sin estilo

        Elimina el estilo list-style por defecto pero mantiene el margen en los elementos de las listas secundarias anidadas. Si quisiéramos que esas segundas listas no tuvieran viñetas ni margen, habría que añadir la clase en esa segunda lista.

        • Consectetur adipiscing elit
        • Integer molestie lorem at massa
        • Facilisis in pretium nisl aliquet
        • Nulla volutpat aliquam velit

        En línea

        Elimina las viñetas de la lista y aplica un ligero margin con la combinación de los clases: .list-inline y .list-inline-item.
