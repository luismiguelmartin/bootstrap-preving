      Layouts

      Bootstrap 4 dispone de un modelo simplificado (a la vez que ampliado) de cuadrícula flexible para maquetación (https://getbootstrap.com/docs/4.1/layout/grid/).

      El sistema de retícula (o cuadrícula) de Bootstrap utiliza una serie de contenedores (.container o .container-fluid), filas (.rows) y columnas (.col, .col-sm, col-lg-2, .col-6,…) para disponer y alinear el contenido de manera totalmente responsive.

      Ejemplos

      Estos fundamentos de maquetación en la retícula se aplican a casi todo el desarrollo de elementos front: formularios, paneles, etc. pudiéndose anidar, combinar y ajustar según se necesite.
